# Astro Starters

Welcome to the Astro Starters collection! Here you'll find a variety of starter templates designed to help you quickly set up your Astro projects. Whether you're building a blog or portfolio, there's a starter template to suit your needs.


## **Launchpad**

![Launchpad screenshot](https://raw.githubusercontent.com/roicort/launchpad/main/public/SS-1.png)

**What it is:** Astro Launchpad — modern scaffolding for agencies and studios with a focus on content-driven sites and composable components.

**Features:**
- MDX blog with featured posts and tag listings
- Typed content collections (posts, authors, socials)
- Pagefind site-wide search with accessible modal
- SEO-ready (OpenGraph/Twitter), RSS and sitemap generation
- Theme variants (light/dark/blue) with persistent toggle
- Deploy buttons for Railway / Netlify / Vercel

**Quick start:** `bun install` → `bun run dev`  ·  [Code](https://github.com/roicort/launchpad) · [Demo](https://roicort.github.io/launchpad/) · License: BSD-3-Clause

## **Dasein**

![Dasein screenshot](https://raw.githubusercontent.com/roicort/dasein/main/public/SS-1.png)

**What it is:** A personal website and portfolio starter built with TypeScript and Tailwind, optimized for content and visual presentation.

**Features:**
- Blog with Markdown/MDX, featured post and tags
- Typed collections and content schemas
- Pagefind search and accessible modal
- SEO, RSS and sitemap support
- Theme support with persistent toggle and debug borders

**Quick start:** `bun install` → `bun run dev`  ·  [Code](https://github.com/roicort/dasein) · [Demo](https://roicort.github.io/dasein/) · License: AGPL-3.0

## **Typewriter**

![Typewriter screenshot](https://raw.githubusercontent.com/roicort/typewriter/main/public/SS-1.png)

**What it is:** A minimal, writer-focused blog starter with MDX, typed collections, and a clean editorial layout.

**Features:**
- MDX and Markdown blog support with typed schemas
- Pagefind search and accessible modal
- SEO (OpenGraph/Twitter), RSS and sitemap
- Themes with persistent toggle and simple customization

**Quick start:** `bun install` → `bun run dev`  ·  [Code](https://github.com/roicort/typewriter) · [Demo](https://roicort.github.io/typewriter/) · License: AGPL-3.0

## **Said Yes!**

**What it is:** A small Astro theme for wedding invitations and event pages, demonstrating GSAP-driven animations and event components.

**Highlights:**
- Event-oriented templates and components
- GSAP animations for polished interactions
- Simple content structure to add event info and RSVPs

**Quick start:** `bun install` → `bun run dev`  ·  [Code](https://github.com/roicort/saidyes) · [Demo](https://roicort.github.io/saidyes/) · License: AGPL-3.0
